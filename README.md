 💰 Personal Expense Tracker (Version 1.0)

📖 Overview

Personal Expense Tracker is a Python-based terminal application that helps users record and manage their daily and monthly expenses. The application stores data in JSON files, allowing expenses to persist even after the program is closed.

This project was developed as part of my Python learning journey to improve my skills in file handling, JSON, exception handling, and building practical software.

---

✨ Features

- 💵 Record monthly expenses
- 🪙 Record daily expenses
- 📅 Automatically stores expenses by date
- 📂 JSON-based data storage
- 💱 Custom currency symbol support
- 📊 View previous daily and monthly expense records
- 🛡️ Basic error handling for missing or invalid JSON files

---

🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- Exception Handling
- datetime Module
